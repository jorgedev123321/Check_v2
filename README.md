# Academic Environment Validator (v2.4.2)

## Propósito
Esta herramienta asegura que las variables de entorno y los permisos de la terminal sean compatibles con los retos de la plataforma `academic.soqui.dev`.

## Contenido
- `main.c`: Código fuente del validador (v2.4.2-stable).
- `check`: Binario estático optimizado para sistemas Linux x64.
